# magIA · IA aplicada con Machine Learning y Python

Repositorio estático listo para GitHub Pages.

## Contenido
- `index.html`: landing principal.
- `styles.css`: estilos magIA.
- `dashboards/`: dashboards HTML interactivos.
- `notebooks/`: notebooks IPYNB descargables.
- `data/`: bases CSV sintéticas.
- `.nojekyll`: evita que GitHub Pages procese el sitio con Jekyll.

## Publicación
Este sitio puede publicarse desde GitHub Pages usando la rama `main` y la carpeta raíz `/`.
